# Gladys Assistant Add-on for Home Assistant

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FGladysAssistant%2Fhassio-addon-gladys)

## About

This repository contains a Home Assistant add-on for [Gladys Assistant](https://gladysassistant.com/), a privacy-first, open-source home automation software.

## Installation

1. Click the button above, or manually add this repository URL in your Home Assistant instance: **Settings** → **Add-ons** → **Add-on Store** → **⋮** → **Repositories** and add:
   ```
   https://github.com/GladysAssistant/hassio-addon-gladys
   ```
2. Find the **Gladys Assistant** add-on in the store and click **Install**.
3. Configure the add-on options (timezone, port, etc.).
4. Start the add-on.
5. Open the Web UI to access Gladys Assistant.
